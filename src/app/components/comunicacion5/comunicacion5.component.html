COMUNICACIÓN ViewChild y ViewChildren


son decoradores que se utilizan para acceder a elementos del DOM (Document Object Model) o componentes hijos desde un componente padre. La comunicación se puede lograr mediante la obtención de referencias a componentes hijos o elementos del DOM en un componente padre y luego interactuando con ellos. No se suele utilizar esto para comunicacion entre componentes lo normal es usar un servicio, RxJs entro otros. Para este caso usaremos ViewChild y ViewChildren para la comunicacion entre un padre y un hijo
